Data Understanding#

TAHAP DATA UNDERSTANDING#

Dalam proses data mining, Data Understanding merupakan fase awal yang berfokus pada pengenalan dataset secara menyeluruh. Pada tahap ini, peneliti tidak langsung melakukan analisis, tetapi terlebih dahulu mempelajari struktur, isi, serta kondisi data yang tersedia.

Pada penelitian ini, proses pemahaman data dilakukan terhadap atribut petal pada dataset Iris.

1. Tahap Akuisisi Data#

Langkah pertama adalah memperoleh dataset yang akan dianalisis. Dataset yang digunakan adalah Iris Dataset dalam format CSV dengan nama file IRIS.csv.

Iris Dataset merupakan dataset publik yang sering digunakan dalam pembelajaran klasifikasi dan machine learning karena strukturnya sederhana namun representatif.

Karakteristik umum dataset:

  • Total 150 sampel bunga

  • Terdiri dari tiga spesies:

    • Iris-setosa

    • Iris-versicolor

    • Iris-virginica

  • Memiliki lima atribut:

    • sepal_length

    • sepal_width

    • petal_length

    • petal_width

    • species

Pada tahap ini dilakukan pemeriksaan awal seperti:

  • Memastikan file berhasil dibaca sistem

  • Mengecek jumlah baris dan kolom

  • Mengidentifikasi tipe data masing-masing atribut

Tahapan ini memastikan data dalam kondisi siap untuk dianalisis.

2. Pemaparan Struktur Data#

Setelah data berhasil dimuat, dilakukan peninjauan terhadap struktur dataset.

Detail dataset:

  • 150 baris data

  • 5 kolom atribut

Tipe atribut:

  • Empat atribut numerik (float):

    • sepal_length

    • sepal_width

    • petal_length

    • petal_width

  • Satu atribut kategorikal:

    • species

Setiap baris merepresentasikan satu bunga Iris.

Dataset tidak mengandung missing value sehingga tidak diperlukan proses pembersihan data pada tahap ini.

Makna masing-masing atribut:

  • sepal_length → panjang kelopak (cm)

  • sepal_width → lebar kelopak (cm)

  • petal_length → panjang mahkota (cm)

  • petal_width → lebar mahkota (cm)

  • species → klasifikasi jenis bunga

Tahap ini bertujuan agar peneliti memahami konteks dan arti setiap variabel sebelum analisis dilakukan.

3. Eksplorasi Awal Dataset#

Eksplorasi dilakukan untuk memperoleh gambaran umum serta menemukan pola awal dalam data.

Beberapa aktivitas yang dilakukan meliputi:

  • Mengamati distribusi nilai pada setiap atribut numerik

  • Membandingkan ukuran bunga antar spesies

  • Mengidentifikasi perbedaan karakteristik masing-masing kelas

  • Menghitung statistik deskriptif (mean, median, nilai minimum, maksimum, dan standar deviasi)

  • Membuat visualisasi seperti histogram, boxplot, dan scatter plot

Melalui proses ini, dapat terlihat potensi pemisahan kelas berdasarkan atribut tertentu serta hubungan antar variabel.

4. Evaluasi Kualitas Data#

Sebelum melanjutkan ke tahap analisis lanjutan, dilakukan pengecekan kualitas data.

Langkah-langkah verifikasi:

  • Memastikan tidak ada nilai kosong

  • Mengecek kemungkinan data duplikat

  • Memastikan tipe data sudah sesuai

  • Memastikan jumlah data tetap 150 baris

  • Memastikan tidak terjadi kesalahan saat proses pembacaan file

Tahap ini penting untuk menjaga validitas hasil analisis sehingga kesimpulan yang diperoleh dapat dipercaya.

5. Eksplorasi iris menggunakan orange#

A

Iris dataset:

  1. File → Mengimpor dataset Iris (150 data, 4 atribut numerik, 1 kelas).

  2. Select Columns → Menentukan fitur (sepal & petal) dan target (species).

  3. Data Table → Melihat isi dan struktur data.

  4. Feature Statistics → Menampilkan statistik deskriptif (mean, min, max, dll).

  5. Rank → Mengetahui fitur paling berpengaruh (biasanya petal paling tinggi).

  6. Scatter Plot → Melihat hubungan antar variabel dan pemisahan kelas.

  7. Box Plot → Melihat perbedaan distribusi tiap spesies.

  8. Correlations → Melihat hubungan antar atribut numerik.

  9. Pivot Table → Mengecek jumlah data tiap kelas (50-50-50).

B

Dataset terdiri dari 150 data dengan 4 atribut numerik (sepal length, sepal width, petal length, petal width) dan 1 atribut kategorikal sebagai target (iris). Tidak terdapat missing value sehingga data siap untuk dianalisis lebih lanjut.

C

Pada tahap eksplorasi data menggunakan aplikasi Orange, digunakan dataset Iris sebagai bahan analisis. Berdasarkan tampilan pada widget Data Table, dataset ini memiliki 150 baris data (instances) dan 5 kolom variabel. Seluruh data dalam dataset ini lengkap tanpa adanya nilai yang kosong (missing value), sehingga dapat langsung digunakan untuk proses analisis.

Selain itu, terdapat satu variabel kategorikal yaitu species (atau iris) yang menunjukkan jenis bunga iris, terdiri dari tiga kelas: Iris-setosa, Iris-versicolor, dan Iris-virginica.

Setiap baris pada tabel merepresentasikan satu sampel bunga iris dengan ukuran masing-masing bagian bunga serta label jenisnya. Dataset ini sangat populer dalam pembelajaran machine learning, khususnya untuk tugas klasifikasi, karena strukturnya sederhana, bersih, dan tidak memiliki data yang hilang.

D

Pada bagian Column Statistics, ditampilkan ringkasan statistik deskriptif dari setiap variabel pada dataset Iris. Statistik ini memberikan gambaran awal mengenai karakteristik data sebelum dilakukan proses analisis atau pemodelan.

Beberapa informasi yang ditampilkan antara lain:

  • Mean (rata-rata) Menunjukkan nilai rata-rata dari seluruh data pada setiap variabel. Mean membantu mengetahui kecenderungan umum nilai suatu fitur. Misalnya, rata-rata sepal length berada di sekitar 5–6 cm, yang menunjukkan ukuran umum panjang sepal bunga iris.

  • Median (nilai tengah) Merupakan nilai yang berada tepat di tengah setelah data diurutkan dari yang terkecil hingga terbesar. Median berguna untuk melihat pusat data yang lebih stabil, terutama jika terdapat nilai ekstrem (outlier).

  • Mode (nilai yang paling sering muncul) Nilai yang memiliki frekuensi kemunculan tertinggi. Mode membantu mengetahui nilai yang paling umum atau dominan dalam suatu variabel.

  • Nilai Minimum dan Maksimum Menunjukkan batas bawah dan batas atas dari data. Dari nilai ini dapat diketahui rentang (range) data, sehingga kita bisa memahami variasi ukuran setiap fitur.

  • Dispersion (tingkat penyebaran data) Menggambarkan seberapa besar variasi atau keragaman data dalam suatu variabel. Semakin besar nilai dispersion, semakin menyebar data tersebut. Pada hasil yang terlihat, variabel petal length dan petal width memiliki tingkat penyebaran yang lebih besar dibandingkan variabel sepal. Hal ini menunjukkan bahwa ukuran petal lebih bervariasi antar spesies.

  • Missing Value (data kosong) Menunjukkan jumlah data yang tidak memiliki nilai. Dari hasil yang ditampilkan, semua variabel memiliki 0% missing value, artinya dataset lengkap dan tidak memerlukan proses pembersihan data tambahan.

E

Pada bagian Rank, dilakukan pemeringkatan variabel berdasarkan tingkat kepentingannya dalam membedakan jenis bunga iris menggunakan metode Information Gain Ratio dan Gini Decrease.

Hasilnya menunjukkan urutan: petal_length, petal_width, sepal_length, dan sepal_width.

Variabel petal_length dan petal_width memiliki nilai tertinggi, sehingga paling berpengaruh dalam membedakan tiga spesies iris. Sementara itu, sepal_width memiliki nilai terendah dan kontribusinya relatif kecil.

Berdasarkan hasil tersebut, saya memilih menggunakan variabel petal (petal_length dan petal_width) karena keduanya paling efektif dalam proses klasifikasi.

F

Pada tahap visualisasi data menggunakan fitur Scatter Plot di Orange, ditampilkan perbandingan antara dua variabel, yaitu petal_length (panjang mahkota) pada sumbu X dan petal_width (lebar mahkota) pada sumbu Y.

Setiap titik pada grafik merepresentasikan satu data bunga iris. Warna yang berbeda digunakan untuk membedakan jenis bunga (species), yaitu Iris-setosa, Iris-versicolor, dan Iris-virginica.

Dari grafik terlihat bahwa data Iris-setosa membentuk kelompok yang sangat terpisah dari dua jenis lainnya. Iris-setosa memiliki nilai petal_length dan petal_width yang paling kecil, sehingga titik-titiknya terkumpul di bagian kiri bawah grafik dan tidak bercampur dengan spesies lain.

Sementara itu, Iris-versicolor berada di bagian tengah grafik dengan ukuran petal sedang. Iris-virginica berada di bagian kanan atas dengan ukuran petal yang paling besar. Kedua spesies ini terlihat cukup berdekatan, namun masih dapat dibedakan karena virginica memiliki nilai yang cenderung lebih tinggi.

Terlihat bahwa variabel petal_length dan petal_width mampu memisahkan ketiga spesies dengan sangat jelas, terutama Iris-setosa yang terpisah sempurna. Visualisasi ini memperkuat hasil analisis sebelumnya bahwa ukuran mahkota bunga (petal) merupakan variabel yang paling efektif dalam membedakan jenis bunga iris.

G

Pada tampilan Box Plot, ditunjukkan penyebaran data untuk variabel petal_length (panjang mahkota bunga) secara keseluruhan.

Grafik ini memperlihatkan nilai minimum, maksimum, median, serta rata-rata dari seluruh data. Terlihat bahwa rata-rata panjang petal adalah sekitar 3.759, dengan median sekitar 4.350. Sebagian besar data berada pada rentang kurang lebih 1.6 hingga 5.1, sedangkan nilai terendah mendekati 1.0 dan nilai tertinggi mendekati 7.0.

Box Plot membantu kita memahami bagaimana data tersebar dan seberapa besar variasinya. Dari grafik terlihat bahwa panjang petal memiliki rentang yang cukup lebar, menunjukkan variasi ukuran yang cukup besar dalam dataset Iris. Hal ini menandakan bahwa variabel petal_length memiliki karakteristik yang beragam dan informatif untuk analisis lebih lanjut.

H

Pada tahap analisis menggunakan fitur Correlations di Orange, dilakukan perhitungan hubungan antar variabel numerik pada dataset Iris menggunakan metode Pearson correlation. Metode ini digunakan untuk mengukur seberapa kuat hubungan linear antara dua variabel.

Nilai korelasi berada pada rentang -1 sampai +1:

  • Nilai mendekati +1 menunjukkan hubungan positif yang kuat (jika satu variabel naik, variabel lain juga naik).

  • Nilai mendekati -1 menunjukkan hubungan negatif (jika satu variabel naik, variabel lain turun).

  • Nilai mendekati 0 berarti hubungan antar variabel lemah atau hampir tidak ada hubungan.

Berdasarkan hasil yang terlihat:

  • petal_length dan petal_width memiliki korelasi sangat kuat (+0.963). Artinya, semakin panjang mahkota bunga, semakin lebar juga mahkotanya.

  • petal_length dan sepal_length memiliki korelasi kuat (+0.872).

  • petal_width dan sepal_length juga menunjukkan korelasi kuat (+0.818).

  • petal_length dan sepal_width memiliki korelasi negatif sedang (-0.421).

  • petal_width dan sepal_width memiliki korelasi negatif lemah hingga sedang (-0.357).

  • sepal_length dan sepal_width memiliki korelasi sangat lemah (-0.109), sehingga hampir tidak memiliki hubungan yang berarti.

Dari hasil ini dapat disimpulkan bahwa variabel petal (panjang dan lebar) memiliki hubungan yang paling kuat dibandingkan variabel sepal. Hal ini menunjukkan bahwa fitur petal lebih saling berkaitan dan cenderung lebih informatif dalam analisis, seperti untuk klasifikasi atau pemilihan fitur pada dataset Iris.

Melalui analisis korelasi ini, kita dapat mengetahui variabel mana yang memiliki hubungan kuat dan mana yang lemah, sehingga membantu dalam menentukan fitur yang paling relevan untuk analisis lanjutan.

Pada bagian Pivot Table, ditampilkan jumlah data untuk setiap jenis bunga.

Terlihat bahwa:

  • Iris-setosa berjumlah 50 data

  • Iris-versicolor berjumlah 50 data

  • Iris-virginica berjumlah 50 data

  • Total seluruh data adalah 150

Hal ini menunjukkan bahwa dataset Iris memiliki jumlah data yang seimbang untuk setiap jenis bunga. Data yang seimbang ini sangat baik untuk proses analisis dan klasifikasi karena tidak ada kelas yang jumlahnya lebih dominan.

# Statistik deskriptif

import pandas as pd
from scipy import stats

df = pd.read_csv("IRIS.csv")

# Ambil hanya kolom numerik (tanpa species)
df_numeric = df.select_dtypes(include=['int64','float64'])

for col in df_numeric.columns:
    print("===== Statistik untuk", col, "=====")
    print("Jumlah Data      :", df[col].count())
    print("Rata-rata        :", df[col].mean())
    print("Nilai Minimum    :", df[col].min())
    print("Q1               :", df[col].quantile(0.25))
    print("Median (Q2)      :", df[col].quantile(0.5))
    print("Q3               :", df[col].quantile(0.75))
    print("Nilai Maximum    :", df[col].max())
    print("Kemencengan      :", round(df[col].skew(), 6))

    mode = stats.mode(df[col], keepdims=True)
    print("Modus            :", mode.mode[0], "dengan jumlah", mode.count[0])

    print("Standar Deviasi  :", round(df[col].std(), 6))
    print("Variansi         :", round(df[col].var(), 6))
    print("\n")
